DML: Data Manipulation Language, comandos utilizados p/manipular os dados armazenados em uma ou
     mais tabelas de um BD. Atravez de agrupamento de comandos, é possível  realizar todas as 
     operações chamada de CRUD (Creat, Research, Update and Delete).

  - INSERIR registros em uma tabela:
	- É fácil, pois é regrado (tem sempre um molde)
		- INSERT INTO (nome_da_table)
		  (nomecoluna1,nomecoluna2,nomecoluna3) VALUE (value1,value2,value3)
